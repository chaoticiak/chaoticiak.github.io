All PuzzleScript games I've created:

  • 12345: My first ever game, about merging blocks.
  • Repel: I saw a game with these mechanics but poor level design, so I set out to make my own game with better levels. To some extent.
  • Rock, Paper, Scissors: The PuzzleScript game I'm most proud of, with interesting interaction between elements. That said, it's pretty old by now and I'd revisit it with new levels if I had time.
  • JAM3 Game: A game I made just to show match-3 games can be puzzly.
